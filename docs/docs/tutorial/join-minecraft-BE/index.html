MCサーバーへの入り方 ～BE編～

おそらく、Discordサーバーへの入り方を見てDiscordサーバーに入れたと思います。
そしたらもうMinecraftサーバーに入ったのと同然です！ 下の入り方を見て、BattleEarthに入りましょう。
また、これはBE版の入り方です。Java版はこちらをご覧ください。

①Minecraftを起動​

まずはこれをしなきゃ始まりません！

②遊ぶをクリック​

起動ができたら遊ぶを選択します。当たり前ですよね！（笑）

③サーバーをクリック​

上にあるサーバーをクリックします。

④サーバーを追加をクリック​

続いてはサーバーを追加します。この追加するボタンは一番下にあります。

⑤IPを入力​

続いてはIPを入力します。IPはDiscordサーバーの#🌐¦ꓲꓑアドレス に書かれています。サーバー名は任意の名前でokです。ポートはそのままです。

外部サーバーを追加
サーバー名：BattleEarth（任意）
サーバーアドレス：#🌐¦ꓲꓑアドレス のIPアドレス
ポート：19132

設定が完了したら完了をクリックしてサーバー情報を保存します

⑥サーバーに参加​

いよいよサーバーに参加します。先ほど作成したサーバーを選択してサーバーに参加をクリックします。

完了！！​

これでサーバーへの参加は完了です。Earthサーバーに移動するまでお待ちください。
また、Geyserとクライアントのバージョンが合わないと参加できないので、運営が更新するまで参加できない場合があります。その場合はお問い合わせください。
